# Music Visual Companion A2 — Static Direction

## Human-approved objective

Replace the rejected simulated camera and water motion with an intentionally static visual companion. Preserve the existing moving render as review evidence and create a new candidate lineage rather than overwriting it.

## Scope

- Make `static-hold-v1` the default visual render profile.
- Scale and crop the approved source once, then hold the exact frame without displacement, pan, zoom, or synthetic environmental motion.
- Preserve the 12-second review gate, exact audio binding, two-second fade-in, four-second fade-out, and private H.264/AAC output.
- Include the render profile in source-binding identity so a static replacement does not collide with the historical moving visual.
- Expose a static replacement action when a candidate still displays the retired profile.

## Boundary

No generative video model, service, upload, publication, retry loop, or advanced animation is added. More capable visual effects remain a separately researched future slice.
