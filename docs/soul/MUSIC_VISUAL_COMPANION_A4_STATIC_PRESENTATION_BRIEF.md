# Music Visual Companion A4 — Static Presentation

Status: owner-authorized implementation candidate

Authorization date: 2026-07-18

## Outcome

Make an approved Visual Studio still the creative source for a Music Studio
companion. Retire procedural water and other FFmpeg-authored visual effects from
the active path. FFmpeg remains only the bounded encoder and muxer.

## Production workflow

```text
approved Visual Studio candidate
  -> exact Music candidate binding
  -> choose static presentation settings
  -> exact 12-second static encode gate
  -> human framing review
  -> exact full-duration encode + audio mux gate
  -> private review candidate
```

Static settings are part of the exact approval digest:

- full-image `contain` or centered `cover` framing;
- six-digit matte color for `contain` framing;
- zero-to-ten-second intro and outro fades;
- static mode only.

The resulting frame does not move, displace, pan, zoom, or synthesize an
environmental effect. The short MP4 is only a reviewable encoding of the chosen
framing. The full companion repeats that approved static encoding, applies the
approved fades, binds the exact lossless Music candidate, and remains private.

## Generated-motion boundary

Generated motion remains visible but unavailable. It may be enabled only after
Visual Studio A3 qualifies a motion model on the AMD host for compatibility,
resource use, output integrity, cancellation, cleanup, and review behavior.

## Migration

Existing procedural-effect records remain historical evidence and cannot
advance. A current Visual Studio binding is migrated by creating the new
deterministic `static-hold-v2` lineage and then removing the superseded local
record after exact identity verification. Records are not rewritten in place.

## Exclusions

- no procedural water, displacement, camera motion, or generated motion;
- no image or video model invocation in Music Studio;
- no listener, queue, watcher, scheduler, automatic retry, or publication;
- no changes to the selected song candidate or its audio artifacts.
